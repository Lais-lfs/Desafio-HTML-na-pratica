Ferramentas utilizadas

Uma das ferramentas mais comuns para criar páginas na web utilizando HTML, é o Visual Studio Code, que é uma ferramenta gratuita e intuitiva, o que é excelente para quem está começando a aprender.

Esse software pode ser instalado de forma fácil por meio de seu próprio site: https://code.visualstudio.com/

O Visual Studio Code possui várias extensões, desde extensões de temas para que você tenha um ambiente customizado, até mesmo extensões que ajudam você a programar com mais facilidade. Um exemplo disso é a extensão “Live Server” que inicia um servidor local de desenvolvimento no qual você consegue visualizar o seu código e acompanhar as modificações feitas nele apenas salvando o arquivo (ctrl + s) no vscode.

Além disso, o Visual Studio possui extensões já pré-configuradas que podem te ajudar muito a programar de forma rápida e descomplicada.

Inspetor de elementos

O inspetor de elementos é uma ferramenta presente no navegador usada por desenvolvedores.

Estas são algumas das formas que você pode acessar essa ferramenta, sendo que em todas, o navegador deve estar aberto:

  • Apertando a tecla “f12” em seu teclado
  • Clicando com o botão direito mouse > inspecionar
  • Clicando no botão de três pontinhos que fica no canto superior direito do navegador > “Mais ferramentas” > “Ferramentas do programador”

Essa ferramenta possui vários tipos de recursos, como, por exemplo, ter acesso ao html da página em que você está, modificá-lo (mas não serão modificações permanentes), limpar o cache da página, além de ser possível executar comandos gerais de programação.

Estrutura básica do HTML

Antes de entrar de fato na explicação da estrutura do HTML, segue abaixo o passo a passo de como começar, criando o arquivo no Visual Studio Code onde o projeto em HTML será feito:

  • Primeiro crie uma pasta em seu computador, na qual este projeto será salvo;
  • Abra o vscode;
  • Clique em: “Arquivo” > “Abrir pasta” > selecione a pasta que você criou anteriormente;
  • Com a pasta aberta no vscode, clique na opção “Novo arquivo” e dê um nome para o seu projeto
  • Em frente ao nome do seu projeto, digite “.html”, pois assim o vscode entende que este será um arquivo de HTML.

As tags no HTML são compostas por um sinal de maior “<”, o nome da tag e o sinal de menor “>”, desta forma: <body>. E sempre deverão ser fechadas com um “/” antes do nome da tag, assim: </body> (exceto por algumas tags específicas).

O HTML consiste em 3 tags principais que formam sua estrutura básica: html, head e body.



HTML: tag que define uma página HTML, ou seja, tudo que estiver dentro dela será uma página HTML.

Head: dentro da tag head ficam algumas pré-configurações, ou seja, fica tudo aquilo que carrega sem que apareça para o usuário.

Body: é de fato o corpo do projeto, ou seja, tudo aquilo que está dentro dessa tag será exibido visualmente para o usuário.

Atributos de tags

Os atributos são propriedades que uma tag pode ter, sendo de dois tipos: globais ou específicas. Os atributos globais todas as tags vão ter, enquanto que as específicas serão particulares de determinadas tags.

Exemplos de atributos globais:

  • Id: é um atributo identificador, ou seja, cada tag tem um id único, mas todas terão um.
  • Style: é um comando do CSS que pode ser usado para estilizar o HTML, logo, é possível inserir elementos de formatação dentro da tag.

    <strong style=”color: blue”>

  • Class: é uma forma de padronizar a estilização.

    <strong class=”titulo-principal”>

    (nesse exemplo, todo lugar que aparecer “titulo-principal” terá esse estilo configurado.)

Exemplos de atributos específicos:

  • Type: é um atributo que fica dentro do input, logo, ele é específico do input.

    <input type=”number”/>

    <input type=”text”/>

    <input type=”color”/>

  • Atributos src e width na tag img: o img é uma tag para inserir imagens dentro da página, mas ela só de fato funciona se estiver acompanhada do atributo scr:

    <img src=”https://url-da-imagem.jpg”/>

    Além disso, outro atributo do img, é o width, que define a largura da imagem que está sendo inserida:

    <img width=”250” src=”https://url-da-imagem.jpg”/>

Textos

A seguir estão algumas tags específicas de textos, ou seja, tipografia HTML.

  • Tags para títulos, subtítulos e cabeçalhos:
    • <h1> Título
    • <h2> Subtítulo
    • ...
    • <h6> Cabeçalho

  • Tags para serem usadas no meio de textos:
    • Parágrafo: <p>
    • Citação: <blockquote>
    • Itálico: <i>
    • Negrito: <strong>
    • Sublinhado: <u>
    • Elevar elemento: <sub>

Listas ordenadas e não ordenadas

Lista ordenada (ol): faz uma ordenação com números. Para criar listas (ordenadas e não ordenadas) você deve identificar os itens da lista usando a tag <li>.

Exemplo:

    <ol>

        <li>Item 1</li>

        <li>Item 2</li>

        <li>Item 3</li>

    </ol>

  1. Item 1
  2. Item 2
  3. Item 3

Lista não ordenada (ul): não utilizam números mas sim marcadores, por exemplo, bolinhas.

Exemplo:

    <ul>

        <li>Item 1</li>

        <li>Item 2</li>

        <li>Item 3</li>

    </ul>

  • Item 1
  • Item 2
  • Item 3

Link

Links direcionam o usuário para um lugar específico, podendo ser para uma página externa ou na mesma página em que o usuário já está, e para ambos os casos utiliza-se a tag <a>.

Exemplo:

<a href=”https://url-da-pagina.com” target=”_blank”>Nome Clicável</a>

Atributo para a tag <a>:

  • Target:
    • blank: link será aberto em outra aba do navegador
    • self: comportamento padrão
    • parent: não é mais usado
    • top: não é mais usado

  • Title: faz com que apareça um balão com uma mensagem ao passar o mouse sobre o link (atributo que ajuda muito na experiência do usuário dentro do site).
